# hamk-module-machine-vision
This repository created to update weekly course progress and assignments of Machine Vision in ICT Robotics program

# Setup instructions

1. git clone https://github.com/lionwalker/hamk-module-machine-vision.git
2. cd your_repo
3. python -m venv assign1
3. source assign1/bin/activate (or assign1\Scripts\activate on Windows)
4. pip install -r requirements.txt
5. python task-a.py — Save the grid image to outputs folder
6. python task-b2.py — Save the annotated version image to outputs folder
